# FileKit — Browser-Native File Processing Suite

A complete suite of browser-native tools for PDFs and images. Zero servers. Zero tracking. Everything runs directly in your browser.

## ✨ Features

- **🔒 Privacy First** — All files are processed locally in your browser. Nothing is ever uploaded or stored on any server.
- **⚡ Blazing Fast** — No round-trips to servers. Processing happens at native browser speed.
- **🎯 Completely Free** — All tools are free with no limits, ads, or watermarks.
- **🎨 Dark Industrial Design** — Precision tool aesthetic with smooth animations and responsive layout.

## 🛠️ Included Tools

### Image Editor
Apply filters and transformations to images:
- Brightness, Contrast, Saturation adjustments
- Blur effect
- Rotation (0-360°)
- Export as PNG

### Images → PDF Converter
Batch convert multiple images into a single PDF:
- Supports JPG, PNG, WEBP, GIF
- Automatic page sizing and centering
- Progress tracking
- One-click download

### PDF Merger
Combine multiple PDF files into one document:
- Drag-and-drop interface
- Visual file list with ordering
- Powered by pdf-lib
- Progress indication

### PDF Compressor
Reduce PDF file size while maintaining quality:
- Adjustable compression level
- Real-time size comparison
- Shows compression ratio
- Uses pdf-lib optimization

### OCR Scanner (Tesseract.js)
Extract text from images:
- Powered by Tesseract.js
- Real-time progress updates
- Copy extracted text to clipboard
- Support for multiple image formats

### Image Resizer
Resize images for social media and web:
- Custom width/height input
- Aspect ratio lock/unlock
- Built-in social media presets:
  - Instagram Square (1080×1080)
  - Instagram Story (1080×1920)
  - Twitter Post (1200×675)
  - Facebook Cover (851×315)
  - LinkedIn Banner (1584×396)
  - YouTube Thumbnail (1280×720)
